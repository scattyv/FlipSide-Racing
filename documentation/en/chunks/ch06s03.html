6.3. Selecting Racers

To limit the graphed information to a specific racer and see the individual lap information for a racer you must select their name in the summary table on the upper right side of the Statistics Screen. This can only be done if a race is not currently running. When a racer is selected the graph will update with only that racers information and the table on the lower left side of the screen will show the individual lap information for that racer.
