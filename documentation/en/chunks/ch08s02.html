8.2. Device Drivers

All the currently supported lap counting hardware uses the same FTDI Driver for communication. If your computer is not recognizing the hardware you plugged in you may need to install a driver. You can download the latest version of the driver for Windows, Mac and Linux from FTDI's website here, www.ftdichip.com/Drivers/VCP.htm.
