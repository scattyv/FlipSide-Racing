6.5. Generate Report

The Generate Report button will generate a report of the race you have selected in the upper left table. This report will include some race configuration information, your club name, club logo, date and time of the race, summary of the race results and individual lap times. The results will be saved in a directory named after the race name under the reports directory. Within this directory will be a web report, raw XML data and a copy of the logo. After the report is generated by clicking the button the report will then be displayed using your web browser. To learn about customizing the appearance of the race report go to the Customize Race Report section.
