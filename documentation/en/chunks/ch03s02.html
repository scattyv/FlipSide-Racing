3.2. Editing Racers

To edit an existing racer you must click on the racer in the table on the race side of the screen. After click the racer all the information about the racer will be displayed in the fields on the left side of the screen. After you have made changes to the record you must click Save to save the changes.

If you need to change the UID of a racer then first select the racer that needs changed and then if you are using lap counting hardware you use the Scan button the same way as when you create racers. As long as the UID is not already used by another racer the scan will overwrite the existing UID and then you can save the racers information. If you are using the keyboard for lap counting then you can just manually update the UID field.
