10.11. Keyboard Shortcuts

There are a few keyboard shortcuts available to perform actions within the software.

If you want the software to announce the current ranks of all the racers hit the "r" key on the keyboard. You can also start a race by hitting "s", end a race by hitting "e", or pause a race by hitting "p".
