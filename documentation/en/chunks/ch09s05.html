9.5. Changing the Logo on the Splash Screen

When the application first launches there is a splash screen that appears with the Flip Side Racing logo before the main window pops up. This logo is loaded from the FlipSideRacingLogo.jpg file in the theme directory. Overwriting this file with your own logo will cause it to be displayed on the splash screen when the application launches. The image should be 520x190 or smaller in size. If it is larger than that the image will be cut off.
